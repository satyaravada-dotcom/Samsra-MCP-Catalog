How to Get Access to an MCP
All MCPs run through Claude Enterprise. The access path depends on the tier shown on each card.
⚡
Self-Serve via Claude.ai
Enable these directly in Claude.ai settings. All you need is a Samsara Claude Enterprise seat.
1
Open claude.ai → sign in with Samsara Okta SSO
2
Profile icon → Settings → Integrations
3
Find the connector → Connect → OAuth authentication
✓ Gmail, Google Calendar, Drive, Slack, Granola, Lucid, Atlassian, Figma, Zoom, Airtable, Smartsheet
🎫
Submit a Helpdesk Ticket
Post in the appropriate Slack channel with the MCP name and your use case. BizTech will provision access.
1
Post in #help-biztech with the MCP name and use case
2
For all Databricks MCPs, post in #helpdesk-data
3
Access provisioned within 1–2 business days — Slack DM when ready
🎫 NetSuite, Databricks Genie spaces, Glean, Salesforce UAT, Workato connectors, Cloudzero, G2, Mixpanel
📋
Team Approval Required
Sensitive integrations need sign-off from the owning team. Check the card for which team owns it.
1
Post in #help-biztech with MCP name and business justification
2
Request routed to the owning team for approval
3
Owning team approves → BizTech provisions access
📋 Finance Systems, People Analytics spaces, Marketo, restricted Salesforce
🚀
Propose a New MCP
Know of an integration that should be in this catalog? Propose it and BizTech will evaluate it for the roadmap.
1
Post in #help-biztech with the tool name and use case
2
BizTech evaluates feasibility, security requirements, and vendor MCP availability
3
Approved proposals are added to the roadmap and this catalog
This catalog is maintained by the BizTech AI Platform team. All data is sourced from a shared Google Sheet — updates appear within minutes of editing.
